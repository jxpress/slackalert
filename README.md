# slackalert

## 概要

サービスでよく使うslackへの通知を行うトレイト

## 依存関係

```build.sbt
resolvers += "Maven Repository on Github" at "https://jxpress.github.io/mvnrepos/"

libraryDependencies += "net.jxpress" % "slackalert_2.11" % "0.0.3"
```
